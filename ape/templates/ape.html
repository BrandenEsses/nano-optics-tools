APE Fitting

How do I use this?

Input a .csv file that contains the following columns in any order. You can upload this .csv file and select which column to fit. The program will generate a plot of the data along with polynomial fits (linear order up to quartic), along with the best fit parameters in a table below the plot.
Note: the column names in the .csv file much exactly match the following:

  • wavenumber (cm-1)
    The DFG wavelength in wavenumbers (cm-1)
  • dfg_xtal
    DFG crystal position (from APE Carmina)
  • dfg_delay
    DFG delay (from APE Carmina)
  • output (mW)
    DFG output power (from powermeter, mW)
  • OPO_WL (nm)
    OPO wavelength (from APE Carmina, nm)
